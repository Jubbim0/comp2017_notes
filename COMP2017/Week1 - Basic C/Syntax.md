# Basic Control flow
- If/If-else syntax:
```C
if ( <expr> ) 
	<stmnt>
else
	<stmnt>
```
- While syntax:
```c
while ( <expr> ) 
	<stmnt>
```
- For syntax:
```c
for ( <initial-expr>; <boolean-expr>; <continuation-expr> ) 
	<stmnt>
```
- Switch syntax:
```c
switch(...)  
{  
case <const-expr>: <statement-sequence>; case ...: ...  
default: ...  
}
```
